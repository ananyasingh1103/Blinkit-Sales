## Project Summary

Developed an end-to-end **Business Intelligence dashboard** using **Power BI** to analyze Blinkit sales data. The project focuses on **data cleaning, data modeling, DAX calculations, KPI tracking, and interactive dashboard development** to generate actionable business insights.

This project demonstrates hands-on experience in **Data Analytics, Data Visualization, and Business Intelligence reporting**.

---

## Business Problem

Retail businesses require clear visibility into:

* Sales performance across outlet types
* Product category contribution
* Customer preferences
* Geographic sales distribution
* Revenue trends over time

This dashboard provides a centralized analytical solution for performance monitoring and decision-making.
